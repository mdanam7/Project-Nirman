# নির্মাণ · Nirman

** Spatial architecture & construction platform.**  
Visualize structures, estimate costs, explore heritage sites, and connect with artisans — all in one web app.

## ✨ Features

| Section | Description | Status |
|---|---|---|
| 🏛️ **Heritage** | Interactive timeline of Bangladesh's architectural landmarks (Lalbagh Fort, Sixty Dome Mosque, etc.) | ✅ Done |
| 🧊 **Visualizer** | Real-time 3D-style structure configurator — set length, width, height, floors, material | ✅ Done |
| 🧮 **Estimator** | District-wise cost breakdown with material pricing for construction projects | ✅ Done |
| 👷 **Artisan Network** | Browse master craftspeople by craft type, location, and rating | ✅ Done |
| 🏪 **Material Market** | Filter and browse construction materials with live pricing | ✅ Done |
| 🌗 **Dark / Light Theme** | Full theme toggle with persistent state | ✅ Done |
| 🎨 **Generative Canvas** | Animated particle background that reacts to the page | ✅ Done |

## 🛠️ Tech Stack

- **HTML5** — semantic structure, canvas element
- **CSS3** — custom design tokens, glassmorphism, animations, responsive layout
- **Vanilla JavaScript** — no frameworks, no dependencies
- **Font Awesome 6** — icons
- **Google Fonts** — Space Grotesk, Noto Sans Bengali, IBM Plex Mono

No build tools. No npm. Just open `index.html` and it works.

## 🤝 How to Contribute

Contributions are very welcome — this project is actively looking for collaborators!

### Areas that need help

- [ ] **Backend integration** — Connect the estimator and market to a real database (PHP + MySQL preferred)
- [ ] **Map view** — Show heritage sites on an interactive Bangladesh map (Leaflet.js)
- [ ] **User accounts** — Login/signup for artisans and clients
- [ ] **Artisan booking** — Allow users to contact or book artisans directly
- [ ] **Real material prices** — Scrape or API-connect live material costs
- [ ] **Mobile responsiveness** — Improve layout on small screens
- [ ] **Bengali language toggle** — Full বাংলা UI switch
- [ ] **Image uploads** — Let artisans upload portfolio photos

### Steps to contribute

1. **Fork** this repository
2. **Create a branch** — `git checkout -b feature/your-feature-name`
3. **Make your changes**
4. **Commit** — `git commit -m "Add: brief description"`
5. **Push** — `git push origin feature/your-feature-name`
6. **Open a Pull Request** — describe what you changed and why

### Guidelines

- Keep JavaScript vanilla (no jQuery or heavy frameworks)
- Follow the existing CSS token system (`--accent`, `--bg`, `--text`, etc.)
- Test in Chrome and Firefox before submitting
- Comment your code — other contributors should understand it easily
